# المحاسبة المالية في القطاع غير الربحي

## المعايير والمكونات المطبقة

### 1. محاسبة الأموال (Fund Accounting)
- **جدول `funds`**: أصناف مالية (أموال غير مقيدة، مقيدة، أوقاف).
- **ربط سطور القيود بالمال**: الحقل `fund_id` في `journal_lines` يربط كل سطر بصنف مال معين.
- **الاستخدام**: تقارير حسب المال، والتأكد من إنفاق الأموال المقيدة حسب الغرض.

| restriction_type | الوصف |
|------------------|--------|
| unrestricted | أموال غير مقيدة - للعمليات العامة |
| restricted | أموال مقيدة - منح/تبرعات لبرنامج أو غرض محدد |
| endowment | أوقاف - رأس مال لا يُصرف، يُستثمر ويُصرف عائداته |

### 2. تصنيف مراكز التكلفة حسب الوظيفة
- **الحقل `center_type` في `cost_centers`**: برنامج، إداري، جمع تبرعات.
- **الاستخدام**: قائمة الأنشطة حسب الوظيفة (Statement of Activities by Function) لمقارنة مصروفات البرامج بالمصروفات الإدارية ومصروفات جمع التبرعات.

| center_type | الوصف |
|-------------|--------|
| program | مصروفات برامج - أنشطة تحقيق الرسالة |
| administrative | مصروفات إدارية عامة |
| fundraising | مصروفات جمع التبرعات والإعلام |

### 3. الميزانيات التقديرية (Budget)
- **جدول `budgets`**: مبالغ مخططة لكل حساب (وربما مركز تكلفة/فترة) حسب السنة المالية.
- **الاستخدام**: تقرير الموازنة مقابل الفعلي (Budget vs Actual) لمراقبة الالتزام بالميزانية.

### 4. أنواع القيود
- **manual**: قيد يومية عادي.
- **receipt**: سند قبض.
- **payment**: سند صرف.
- **donation**: قيد تبرع (للتتبع والتقارير).
- **grant**: قيد منحة.
- **adjusting**: قيد تسوية.
- **opening**: قيد افتتاحي.
- **closing**: قيد إقفال.

### 5. التقارير الخاصة بالقطاع غير الربحي
- **قائمة الدخل**: إيرادات ومصروفات وصافي الربح/الخسارة.
- **الميزانية العمومية**: أصول، التزامات، حقوق الملكية (صافي الأصول).
- **قائمة الأنشطة حسب الوظيفة**: إجمالي الإيرادات، مصروفات برامج/إدارية/جمع تبرعات، صافي النشاط.
- **ميزان المراجعة**: حسب الحساب مع إمكانية التصفية حسب الفترة والمال.

### 6. دليل الحسابات
- **أنواع الحسابات**: أصول (asset)، التزامات (liability)، حقوق ملكية (equity)، إيرادات (revenue)، مصروفات (expense).
- في القطاع غير الربحي تُفسَّر **حقوق الملكية** كـ **صافي الأصول** (بدون قيود / مقيدة حسب المال إن رُبطت الأموال).

### 7. الإقفال السنوي
- قيد الإقفال ينقل الإيرادات والمصروفات إلى حساب صافي الربح/الأرباح المحتجزة (حقوق ملكية).
- يُنفَّذ تلقائياً عند إقفال السنة المالية.

## إعداد النظام للقطاع غير الربحي

1. **إنشاء أصناف أموال**: من إدارة الأموال (إن وُجدت واجهة) أو من خلال Seeder: مال عام غير مقيد، ثم إضافة أموال مقيدة أو أوقاف حسب الحاجة.
2. **تصنيف مراكز التكلفة**: تعيين كل مركز تكلفة إلى (برنامج، إداري، جمع تبرعات).
3. **إدخال الميزانيات التقديرية**: لكل سنة مالية وحساب/مركز تكلفة حسب الحاجة.
4. **استخدام مركز التكلفة في القيود**: عند إنشاء القيود اختيار مركز التكلفة لتمكين تقرير قائمة الأنشطة حسب الوظيفة.
5. **استخدام الصنف المالي (fund_id)** في سطور القيود عند وجود أموال مقيدة أو أوقاف.
