# معلومات تسجيل الدخول

## 🔐 بيانات تسجيل الدخول الافتراضية

بعد تشغيل `php artisan migrate:fresh --seed`، يمكنك تسجيل الدخول باستخدام:

- **البريد الإلكتروني:** `admin@example.com`
- **كلمة المرور:** `admin123`

## 🔗 رابط تسجيل الدخول

```
http://localhost:8080/login
```

أو

```
http://your-domain.com/login
```

## ⚠️ تحذير أمني

**يجب تغيير كلمة المرور فوراً بعد أول تسجيل دخول!**

يمكنك تغيير كلمة المرور من:
1. تسجيل الدخول إلى لوحة التحكم
2. الذهاب إلى إعدادات الموقع (إذا أضفت هذه الميزة)
3. أو تعديلها مباشرة من قاعدة البيانات

## 🔒 حماية لوحة التحكم

جميع مسارات `/dashboard/*` محمية الآن بنظام المصادقة.

إذا حاولت الوصول إلى لوحة التحكم بدون تسجيل الدخول، سيتم توجيهك تلقائياً إلى صفحة تسجيل الدخول.

## 🚪 تسجيل الخروج

يمكنك تسجيل الخروج من خلال:
- زر "تسجيل الخروج" في القائمة الجانبية للوحة التحكم
- أو زيارة: `/logout`
