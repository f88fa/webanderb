# إعداد قاعدة البيانات MySQL

## الخطوات المطلوبة:

### 1. إنشاء قاعدة البيانات

إذا لم تكن قاعدة البيانات موجودة، قم بإنشائها:

```sql
CREATE DATABASE dashlaravel CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. تحديث ملف `.env`

تأكد من أن ملف `.env` يحتوي على الإعدادات التالية:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=dashlaravel
DB_USERNAME=root
DB_PASSWORD=root
```

**ملاحظة:** إذا كنت تستخدم MAMP، قد يكون المنفذ `8889` بدلاً من `3306`.

### 3. تشغيل Migrations و Seeders

```bash
php artisan migrate:fresh --seed
```

هذا الأمر سيقوم بـ:
- حذف جميع الجداول الموجودة
- إنشاء الجداول من جديد
- إضافة البيانات الافتراضية (الإعدادات والمستخدم الافتراضي)

### 4. بيانات تسجيل الدخول الافتراضية

بعد تشغيل Seeder، يمكنك تسجيل الدخول باستخدام:

- **البريد الإلكتروني:** `admin@example.com`
- **كلمة المرور:** `admin123`

**⚠️ مهم:** قم بتغيير كلمة المرور بعد أول تسجيل دخول!

### 5. تصدير قاعدة البيانات

بعد التأكد من أن كل شيء يعمل، قم بتصدير قاعدة البيانات:

```bash
mysqldump -u root -p dashlaravel > dashlaravel_backup.sql
```

أو من phpMyAdmin:
1. اختر قاعدة البيانات `dashlaravel`
2. اضغط على "تصدير" (Export)
3. اختر "SQL"
4. اضغط "تنفيذ" (Go)

---

## استكشاف الأخطاء

### خطأ: Access denied
- تأكد من اسم المستخدم وكلمة المرور في ملف `.env`
- تأكد من أن MySQL قيد التشغيل

### خطأ: Connection refused
- تأكد من أن MySQL قيد التشغيل
- تحقق من المنفذ (Port) - قد يكون `3306` أو `8889` في MAMP

### خطأ: Unknown database
- تأكد من إنشاء قاعدة البيانات أولاً
- تحقق من اسم قاعدة البيانات في ملف `.env`
